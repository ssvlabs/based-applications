# :space_invader: __Based Applications Contracts ABI__

&nbsp;

## :book: Description

This repository contains the ABI (Application Binary Interface) files for Based Applications Smart Contracts, deployed across testnets and mainnet environments.

&nbsp;

## :mag_right: Finding the Right ABI

`docs/abi/<environment>/<version>/`

&nbsp;

### :arrows_counterclockwise: Examples for Release candidates:

```
docs/abi/dev/v0.0.2-rc.0/abi
docs/abi/dev/v0.0.2-rc.1/abi
docs/abi/dev/v0.1.2-rc.0/abi
```

&nbsp;

### :rocket: Examples for Production releases:

```
docs/abi/testnet/v0.0.1/
docs/abi/mainnet/v1.0.0/
```
